# SB-Swagger

Documentation for Restful APIS

http://localhost:8080/swagger-ui/index.html

@Api(value = "Book Service", description = "My Book Store")    => Controller , we can add description at controller level in documentation
@ApiOperation(value = "store book api")             => Method - instead of method name we can customize                    
@ApiModelProperty(value = "autogenerated id")       => Model  - when opened model data to check what is the type
